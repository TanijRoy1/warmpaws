# 🐾 WarmPaws – Winter Pet Care Services

![Warmpaws Home Page](./src/assets/warmpawsHomePage.png)

**WarmPaws** is a React-based web application designed to help pet owners find and book essential **winter care services** for their pets — from cozy coat fittings to vet consultations.  
Built with modern web technologies, it offers a warm and user-friendly experience for both pet owners and service providers.



## 🌐 Live Demo
🔗 [Live Website](https://warmpaws-b12a9.netlify.app)



## 🎯 Purpose
To provide a one-stop platform for pet owners to ensure their furry friends stay **healthy, comfortable, and protected** during the winter season.



## ✨ Key Features

- 🧥 **Winter Pet Services:** Browse and book from a variety of seasonal pet care services.  
- 👨‍⚕️ **Meet Expert Vets:** View experienced veterinarians with their profiles.  
- 💬 **Testimonials Section:** Real stories from happy pet owners.  
- 📅 **Book Service Form:** Simple booking interface for quick scheduling.  
- 🔐 **User Authentication:** Secure login, registration, and profile management via Firebase.  
- 🧾 **Persistent Layout:** Consistent Navbar and Footer across all pages.  
- 🧠 **Responsive Design:** Fully responsive and optimized for all devices.  
- 🎞️ **Smooth Animations:** Enhanced UX with Swiper, AOS, and Animate.css transitions.

## ⚙️ Installation & Local Setup
### Frontend Setup

1. Clone the frontend repository:
```js
git clone https://github.com/TanijRoy1/warmpaws.git
```

2. Navigate to the project folder:
```js
cd warmpaws
```

3. Install dependencies:
```js
npm install
```

4. Create a `.env` file in the root and add Firebase config.

5. Start the development server:
```js
npm run dev
```

6. The app will run on http://localhost:5173
